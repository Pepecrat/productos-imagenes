# Repositorio Centralizado de Imágenes de Productos

Imágenes de productos servidas via jsDelivr CDN.

## Uso

```
https://cdn.jsdelivr.net/gh/TU_USUARIO/productos-imagenes@main/images/{codigo}.jpg
```

## Agregar nueva imagen

1. Renombrar a `CODIGO.jpg` (ej: `99999.jpg`)
2. Copiar a `/images/`
3. Push a GitHub
4. El manifest se actualiza automáticamente
